# LED Loop Media Checklist

## Use

The LED wall is a looped media / brand atmosphere surface. It is not an interactive website screen.

## Needed Media

| Item | Duration | Format | Resolution | Status | Notes |
| --- | --- | --- | --- | --- | --- |
| XUAN logo / dynamic background loop | 1-2 min | MP4 | TBD | Needed | Brand atmosphere loop. |
| Project case reel | 10-20 clips | MP4 | TBD | Needed | Project montage for local playback. |
| Exhibition holding screen | TBD | PNG / MP4 | TBD | Optional | Stable fallback for setup time. |

## Rules

- Do not load full LED videos on website homepage.
- Use covers and links only if the media needs to appear on the web.
- Keep local playback files separate from public web assets unless cleared.
- Confirm final LED resolution with the build team before export.
